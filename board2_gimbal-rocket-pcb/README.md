# Gimbaling Rocket Engine PCB — Fusion 360 Model

This PCB was designed for a model rocket featuring a gimbaling engine. Although the schematic and layout files were lost during a Windows corruption incident, the full Fusion 360 3D model was preserved. The model includes detailed mechanical representations used for fit‑checking and integration with the gimbal assembly.

---

## Purpose
The board was created to:
- Fit within the gimbal housing  
- Align with servos and linkages  
- Validate mechanical clearances  
- Provide a realistic reference for wiring paths and mounting  

This was my second PCB ever, and my first time integrating electronics with a moving mechanical system.
