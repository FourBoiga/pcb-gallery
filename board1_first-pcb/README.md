# First PCB — Early Prototype

This was the earliest PCB I designed, created before I fully understood layout workflows. The schematic was completed, but the layout was still in progress when the project was interrupted. Even though the board was never fabricated, it represents the starting point of my hardware design journey.

---

## Overview
This board was intended as a simple test platform to learn:
- Schematic capture
- Component selection
- Power routing concepts
- Basic PCB layout strategy

The schematic was fully completed, however due corruption I was only able to recover an older version from before I had started making the layout.

## What I Learned
